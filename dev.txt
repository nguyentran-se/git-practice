# Git flow
- [feat/...] git checkout dev
- [dev] git pull
- [dev] `git checkout -b feat/LMSC11`
- `git checkout -b feat/LMSC-11`
- git commit -m 'feat: something'
  - git push
- git commit --amend --no-edit 
  - git push -f
